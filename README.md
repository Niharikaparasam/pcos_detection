This PCOS Detection project utilizes a machine learning model trained on relevant medical parameters to predict the likelihood of PCOS. The backend is built using Flask, handling API requests for inference. The model is trained using supervised learning techniques, leveraging algorithms like Random Forest, SVM, or XGBoost (depending on the final implementation) with feature selection techniques applied for improved accuracy. The dataset includes key health indicators such as BMI, menstrual cycle irregularities, hormone levels, and symptoms like hair growth and skin darkening. The frontend is built with React (or another framework), providing an intuitive interface for users to input data, which is sent to the backend via REST API (http://localhost:5000/predict). The model processes the data and returns a prediction, indicating whether PCOS is detected or not. Future improvements include cloud deployment, enhanced model accuracy, and additional diagnostic features.
